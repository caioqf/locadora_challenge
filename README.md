# Locadora de veículos

## Requisitos

É necessário ter o Docker e docker-compose instalado e configurado na máquina.

## Rodar

- Clonar o repositorio
- Entrar na pasta do projeto
- Rodar o comando abaixo:

> $ docker-compose up
